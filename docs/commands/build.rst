********************************************************************************
build
********************************************************************************

::

    Initialize and build an index. Shorthand for running an init,add,commit command sequence.
    
    Usage:
      ddb build [args] [PATHS]
    
          --paths arg  Paths
      -h, --help       Print help
          --debug      Show debug output
