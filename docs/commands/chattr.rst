.. _chattr_command:

********************************************************************************
chattr
********************************************************************************

::

    Manage database attributes
    
    Attributes:
    	public	mark database as publicly accessible
    
    
    Usage:
      ddb chattr [+-attribute] [args]
    
      -w, --working-dir arg  Working directory (default: .)
      -h, --help             Print help
          --debug            Show debug output
