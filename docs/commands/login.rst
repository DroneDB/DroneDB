.. _login_command:

********************************************************************************
login
********************************************************************************

::

    Authenticate with a registry.
    
    Usage:
      ddb login [args]
    
          --server arg    Registry server to authenticate to (default:
                          index.dronedb.app)
      -u, --username arg  Username
      -p, --password arg  Password
      -h, --help          Print help
          --debug         Show debug output
