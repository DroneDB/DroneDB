********************************************************************************
status
********************************************************************************

::

    Show files and directories index status compared to the filesystem
    
    Usage:
      ddb status [directory] [args]
    
      -w, --working-dir arg  Working directory (default: .)
      -h, --help             Print help
          --debug            Show debug output
