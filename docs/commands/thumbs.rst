.. _thumbs_command:

********************************************************************************
thumbs
********************************************************************************

::

    Generate thumbnails for images and rasters
    
    Usage:
      ddb thumbs output/ *.JPG [args]
    
      -i, --input arg   Files to generate thumbnails of
      -o, --output arg  Output directory where to store thumbnails
      -s, --size arg    Size of the largest side of the images (default: 512)
          --use-crc     Use CRC for output filenames
      -h, --help        Print help
          --debug       Show debug output
