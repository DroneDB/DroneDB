.. _geoproj_command:

********************************************************************************
geoproj
********************************************************************************

Project images to georeferenced rasters

::

    $ ddb geoproj output/ *.JPG [args]

::

  -o, --output arg  Output path (file or directory)
  -i, --images arg  Images to project
  -s, --size arg    Output image size (size[%]|0) (default: 100%)
